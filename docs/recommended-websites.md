# Recommended Football Data, Analytics, and Research Websites

This page lists reliable websites for football analytics research, tournament context, reproducible data work, and model evaluation. These resources support analysis and documentation; they should not be presented as betting advice.

| Website | Recommended Use |
|---|---|
| [FIFA](https://www.fifa.com/) | Official tournament, team, and competition information. |
| [FIFA World Cup 2026](https://www.fifa.com/en/tournaments/mens/worldcup/canadamexicousa2026) | Official FIFA page for the 2026 World Cup. |
| [FBref](https://fbref.com/) | Football statistics tables and historical match data references. |
| [StatsBomb Open Data](https://github.com/statsbomb/open-data) | Open football event data repository for research and examples. |
| [Kaggle Datasets](https://www.kaggle.com/datasets) | Public dataset platform for reproducible data experiments. |
| [scikit-learn Documentation](https://scikit-learn.org/stable/) | Official machine learning library documentation. |
| [pandas Documentation](https://pandas.pydata.org/docs/) | Official documentation for Python data analysis workflows. |
| [NumPy Documentation](https://numpy.org/doc/) | Official numerical computing documentation for Python. |
| [Matplotlib Documentation](https://matplotlib.org/stable/) | Official plotting documentation for reproducible charts. |

## Review Policy

Recommended websites should be reviewed periodically. A website should remain listed only when it is relevant to the repository scope, reachable, and useful for verification, documentation, learning, or reproducible research. Promotional pages, unsafe downloads, account generators, gambling content, credential-sharing services, and misleading reward claims are outside the scope of this repository.
