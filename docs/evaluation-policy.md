# Evaluation Policy

Every model experiment should be evaluated with clear assumptions and reproducible steps. Results are research outputs, not guarantees.

| Item | Requirement |
|---|---|
| Dataset version | State the source and snapshot date. |
| Target | Define exactly what the model predicts. |
| Split | Explain train/test or cross-validation logic. |
| Metrics | Include appropriate classification or probabilistic metrics. |
| Limitations | Document uncertainty and known weaknesses. |
