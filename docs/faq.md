# FAQ

## Is this a betting prediction tool?

No. This project is for educational and analytical research. Model outputs are experimental and should not be used for betting or financial decisions.

## What data sources are used?

The project uses example datasets and publicly available football statistics. See `docs/recommended-websites.md` for reference data sources.

## How accurate are the predictions?

Baseline models are intentionally simple and should not be treated as accurate forecasts. Every experiment report should document limitations and uncertainty.
