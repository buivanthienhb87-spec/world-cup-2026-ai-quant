# Security Policy

This repository does not accept content that promotes account compromise, unauthorized access, cheating, credential theft, malware, or abuse of online services. If you discover a security issue in project files, please open a private report through GitHub's security reporting workflow when available, or create a minimal public issue that avoids exposing sensitive details.

Do not publish tokens, passwords, cookies, private keys, or personal data in issues, pull requests, examples, or documentation.
